Get Ricgraph

To get Ricgraph, go to the Ricgraph GitHub repository.

If you like Ricgraph, please give it a GitHub star by clicking on the top right Star button on the Ricgraph GitHub repository page. If you have any suggestions or improvements, please let me know by creating a GitHub Issue.

How to use Ricgraph

  • Install and configure Ricgraph.
  • Start harvesting data, see Ricgraph harvest scripts.
  • Use Ricgraph Explorer, the exploration tool for Ricgraph.
  • Use the Ricgraph REST API, the REST API for Ricgraph.
  • Optional: modify code to fit Ricgraph to your specific use case.
